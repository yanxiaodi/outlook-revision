📸 Visual Installation Guide

Follow these illustrated steps to install Outlook ReVision in minutes

Step 1

Open Outlook Web

Navigate to Outlook on the web and click on the Apps icon in the sidebar to access the add-ins menu.

  • Works on any browser
  • No software installation required
  • Access your emails securely
Step 2

Navigate to Get Add-ins

Click on "Get Add-ins" from the apps menu to open the Office Add-ins store.

💡 Tip: You can also access this from the ribbon menu at the top of Outlook.
Step 3

Go to My Add-ins

In the Add-ins dialog, click on the "My add-ins" tab to view and manage your custom add-ins.

  • View installed add-ins
  • Add custom add-ins
  • Manage add-in settings
Step 4

Add from File

Scroll down and click "+ Add a custom add-in" then select "Add from file..." to upload the manifest.

📋 Note: Make sure you've downloaded the manifest.xml file from the button above before this step.
Step 5

Upload Manifest File

Click "Browse" and select the manifest.xml file you downloaded earlier from your computer.

  • Locate the downloaded file
  • Select the manifest.xml
  • Click Open to proceed
Step 6

Confirm Installation

Review the add-in permissions and click "Install" to complete the installation process.

Step 7

Start Using ReVision! 🎉

Congratulations! Outlook ReVision is now installed. You'll see the add-in icon in the Outlook ribbon when composing emails.

  • Compose a new email to see ReVision in action
  • Access AI features from the taskpane
  • Configure your preferences in settings
  • Enjoy AI-powered email assistance
